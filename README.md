# Shop App

A Flutter e-commerce app for browsing and purchasing shoes, built with Provider state management.

## Features
- Product listing with alternating card colors
- Search products by name in real time
- Filter products by brand using chip selection
- Navigate to detailed product page
- Select shoe size before adding to cart
- Add to cart with validation — size must be selected
- Cart management with delete confirmation dialog
- Provider state management for cart across screens
- Responsive layout — grid view on large screens, list view on mobile
- Input validation and user feedback via SnackBar

## Tech Stack
- Flutter
- Dart
- Provider — state management
- Material Design 3

## What I Learned
- Provider pattern for global state management
- Navigation and routing between screens
- LayoutBuilder for responsive UI
- ListView.builder and GridView.builder
- Real time search and filtering with .where()
- Dialog and SnackBar for user feedback
- Widget extraction and project folder structure


## Getting Started
1. Clone the repo
2. Run `flutter pub get`
3. Add shoe images to `assets/images/` folder
4. Run `flutter run`# shop_app

A new Flutter project.
